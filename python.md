# python 笔记

### 1.程序设计语言

+ 机器语言：二进制语言，直接使用二进制代码表达指令，是计算机硬件可以直接识别和执行的程序设计语言。
    
+ 汇编语言：使用助记符与机器语言中的指令一一对应。
  
+ 高级语言：最接近自然语言的计算机程序设计语言。
  

### 2.编译与解释
+ **编译型**：将高级语言的源代码转换成机器语言的目标代码的计算机程序。

+ **解释型**：将高级语言的源代码逐条转换成目标代码同时逐条运行（执行解释）的计算机程序。

### 3.IPO程序编写方法
+ I:input（输入）
+ P:process（处理）
+ O:output（输出）
