### 恒等函数和softmax函数

+ **恒等函数**：恒等函数会将输入按原样输出，对于输入的信息，不加以任何改动地直
接输出

+ **softmax函数**：softmax函数将输入的向量转换成概率分布，即将向量中的每个数字都转换为0到1之间的实数，并且所有数的总和为1

$$
\begin{align*}
y_k &= \frac{\exp(a_k)}{\sum_{i=1}^n \exp(a_i)}
    = \frac{C \exp(a_k)}{C \sum_{i=1}^n \exp(a_i)} \\
   &= \frac{\exp(a_k + \log C)}{\sum_{i=1}^n \exp(a_i + \log C)} \\
   &= \frac{\exp(a_k + C')}{\sum_{i=1}^n \exp(a_i + C')}
\end{align*}
$$

其中$ a_k$为输入信号

```python
def softmax(a):
    c = np.max(a)
    exp_a = np.exp(a - c) # 溢出对策
    sum_exp_a = np.sum(exp_a)
    y = exp_a / sum_exp_a

    return y
 ```

## 神经网络的学习

+ **学习**：从训练数据中自动获取最优权重参数的过程

+ **泛化能力**：处理未被观察过的数据（不包含在训练数据中的数据）的能力

+ **过拟合**：只对某个数据集过度拟合的状态
  
+ **损失函数**：神经网络以某个指标为线索寻找最优权重参数，可以是任何函数但一般用均方误差和交叉熵误差等
  
+ **均方误差**：

$$
E = \frac{1}{2} \sum_{k} (y_k - t_k)^2
$$

k：数据的维度

+ **one-hot表示**：将正确解标签表示为1，其他标签表示为0的表示方法

+ **交叉熵误差**：

$$E = - \sum_{k} t_k \log(y_k) $$

+ **mini-batch学习**：从训练数据中随机选择一部分数据进行学习的方法
